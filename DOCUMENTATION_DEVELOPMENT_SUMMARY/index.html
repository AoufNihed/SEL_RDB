    Building the Documentation¶

    To build and serve the documentation locally:

    1. Install MkDocs and the Material theme: bash pip install mkdocs mkdocs-material

    2. Serve the documentation locally: bash mkdocs serve

    3. Build the static site: bash mkdocs build

    4. Deploy to GitHub Pages: bash mkdocs gh-deploy
